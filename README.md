💘 520 恶搞表白网站（React版）说明文档
📌 项目简介

这是一个“AI智能告白系统”恶搞网页项目。

核心效果：

假装 AI 在分析你的恋爱数据
“我愿意”按钮会逃跑，无法点击
进入“恋爱系统加载动画”
最终反转为整蛊结果
触发多层弹窗 + 满屏恶搞 UI

整体风格：

💖 甜美恋爱外壳 + 🤡 高强度整蛊内核

🚀 如何运行
1️⃣ 创建 React 项目（Vite 推荐）
npm create vite@latest prank520
cd prank520
npm install
npm run dev
2️⃣ 替换代码

找到：

src/App.jsx

删除里面内容，替换为本项目代码。

3️⃣ 启动
npm run dev

浏览器访问：

http://localhost:5173
